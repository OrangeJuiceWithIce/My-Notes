# LLM
## Alpaca
### 数据集
**instruction（必填）**:优质指令。例如“总结以下文章”。
**input（选填）**:任务的输入。例如一篇文章。
**system**:系统提示词，包括模型的基本行为准则、个性特征和能力范围，由开发者设定。例如“你能够支持20万字的输入和输出”
**output**:模型的输出。
**history**:字符串二元组的列表,历史回答。
```
[
  {
        "instruction": "人类指令（必填）",
        "input": "人类输入（选填）",
        "output": "模型回答（必填）",
        "system": "系统提示词（选填）",
        "history": [
          ["第一轮指令（选填）", "第一轮回答（选填）"],
          ["第二轮指令（选填）", "第二轮回答（选填）"]
        ]
  }
]
```
### 超参数
**temperature**:温度越高，softmax越平滑，输出越丰富。
**top-k**:从概率排名前k的词汇中选择。
**top-p**:从概率和为p的词汇中选择。
